1.O que é um vetor em C++? Explique com suas palavras.
É um sistema que organiza os dados
2.Qual a diferença entre declarar um vetor com valor fixo e inicializá-lo com valores específicos?    
Quando declarado o valor fixo, apenas indica a quantidade de espaços que serão reservados. Quando ja é inicializado com os valores específicos, os espaços já são preenchidos.
3.Explique o que significa índice e qual o intervalo válido para um vetor int not[10];
O índice é o número que indica a quantidade de espaços que devem ser reservados na memória. No exemplo citado, o int quer dizer que é um valor inteiro, o not é o nome e o 10 indica
a quantidade de espaços. Como são 10, vai ser do 0 - 9.
